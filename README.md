# Race Microservice :dragon:

De Microservice race gaan we gebruiken voor het bewaren van gegevens van de verschillende rassen.


## Gegevens

In de Raceservice zullen we de volgende parameters gaan opslaan:

```bash
id
name
speed
alignment
size
abilityBonuses
```
